# 機能仕様 - grid-sprite-editor

## 概要

ChatGPT 等のAIが生成したスプライトシート（N×M グリッドの PNG）を修正・編集するための Windows デスクトップアプリ。

主なユースケース：
- 3×3 スプライトシートでコマが隣にはみ出してしまっている箇所を修正
- グリッドの分割位置がずれている場合に手動調整
- 修正後に個別コマとして書き出してゲームエンジンに取り込む

---

## 機能一覧

### F-01: 画像読み込み・保存
- PNG ファイルをドラッグ&ドロップまたはファイルダイアログで開く
- 上書き保存・名前を付けて保存
- 透過（アルファチャンネル）を保持

### F-02: グリッド表示・設定
- N（列数）× M（行数）を数値入力で設定（デフォルト 3×3）
- グリッド線の色・透明度を変更可能
- グリッド表示/非表示の切り替え

### F-03: グリッド線手動調整
- グリッド線をドラッグして位置を移動
- 等分割ではないスプライトシートに対応
- 「等分割にリセット」ボタン

### F-04: リサイズ
- N（または LCM(N,M)）で割り切れるサイズに変換
- プリセット（3×3 向け）:
  - 768 × 768（1コマ 256×256）
  - 1536 × 1536（1コマ 512×512）★推奨
  - 1920 × 1920（1コマ 640×640）
  - 2400 × 2400（1コマ 800×800）
  - 3072 × 3072（1コマ 1024×1024）
- カスタムサイズ入力も可（割り切れない場合は警告）
- リサイズアルゴリズム: Lanczos（高品質）

### F-05: 矩形選択
- マウスドラッグで矩形の選択範囲を作成
- 選択範囲をドラッグで移動
- 選択範囲のハンドルをドラッグして拡縮
- Delete キーで選択範囲を透明化

### F-06: ラッソ選択（自由形状）
- マウスドラッグで自由形状の選択範囲を描く
- 選択範囲をドラッグで移動
- Delete キーで選択範囲を透明化

### F-07: 消しゴム
- ブラシサイズ調整可能
- 塗った箇所を透明にする

### F-08: コマ単位入れ替え
- セルをクリックして選択 → 別のセルをクリックで入れ替え
- または専用ダイアログで入れ替え先を指定

### F-09: アニメーションプレビュー
- 全コマを指定 FPS で連続再生
- 再生/停止/コマ送り/コマ戻し
- FPS 設定（1〜60）
- ループ再生切り替え

### F-10: ズームイン/アウト
- マウスホイールまたはスライダーでズーム
- ズーム率: 25% 〜 800%
- 「実寸表示」「全体表示」ボタン

### F-11: Undo/Redo
- Ctrl+Z / Ctrl+Y（または Ctrl+Shift+Z）
- 履歴数: 最大 50 ステップ
- 編集操作すべてがアンドゥ対象

### F-12: 個別コマエクスポート
- 全セルを個別 PNG として書き出し
- ファイル名: `{元ファイル名}_{行}_{列}.png`（例: `sprite_0_0.png`）
- 出力先ディレクトリを選択

## キーボードショートカット

| キー | 機能 |
|------|------|
| Ctrl+O | ファイルを開く |
| Ctrl+S | 上書き保存 |
| Ctrl+Shift+S | 名前を付けて保存 |
| Ctrl+Z | Undo |
| Ctrl+Y | Redo |
| Ctrl++ / Ctrl+- | ズームイン/アウト |
| Ctrl+0 | 全体表示 |
| Delete | 選択範囲を削除 |
| V | 移動ツール |
| M | 矩形選択ツール |
| L | ラッソ選択ツール |
| E | 消しゴムツール |
| Space+ドラッグ | キャンバスパン |

## 技術スタック

| 項目 | 選定 | 理由 |
|------|------|------|
| 言語 | Python 3.11+ | 豊富なライブラリ、開発速度 |
| UI | PyQt6 | 高機能、Windows ネイティブ対応 |
| 画像処理 | Pillow | 標準的、扱いやすい |
| 配布 | PyInstaller | exe 化で依存なしに配布可能 |
